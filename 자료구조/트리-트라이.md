# 트리 / 트라이

## 한 줄 요약

> <u>트리는 계층 구조를 표현하는 자료구조이고, 트라이는 문자열 접두사 탐색에 강한 트리다.</u>

트리는 노드와 간선으로 이루어진 계층형 자료구조다.

트라이는 문자열을 저장하고 찾기 위한 트리다. 공통 접두사를 공유해서 저장 효율이 좋다.

## 1. Tree

> <u>트리의 핵심은 사이클이 없고, 루트에서 경로가 유일하다는 점이다.</u>

- 사이클이 없다.
- 루트에서 어떤 노드로 가는 경로는 유일하다.
- 노드가 N개면 간선은 N-1개다.
- 부모-자식 관계로 표현한다.

### 용어

- root: 최상단 노드
- leaf: 자식이 없는 노드
- depth: 루트에서 해당 노드까지의 거리
- height: 해당 노드에서 가장 깊은 리프까지의 거리

### 트리 순회

- 전위 순회: Root -> Left -> Right
- 중위 순회: Left -> Root -> Right
- 후위 순회: Left -> Right -> Root
- 레벨 순회: 위에서 아래, 왼쪽에서 오른쪽

> <u>중위 순회는 BST에서 오름차순 결과가 나온다.</u>

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int data;
    Node* left;
    Node* right;

    Node(int x) : data(x), left(nullptr), right(nullptr) {}
};

void preorder(Node* node) {
    if (node == nullptr) return;
    cout << node->data << ' ';
    preorder(node->left);
    preorder(node->right);
}

void inorder(Node* node) {
    if (node == nullptr) return;
    inorder(node->left);
    cout << node->data << ' ';
    inorder(node->right);
}

void postorder(Node* node) {
    if (node == nullptr) return;
    postorder(node->left);
    postorder(node->right);
    cout << node->data << ' ';
}

void levelorder(Node* root) {
    if (root == nullptr) return;

    queue<Node*> q;
    q.push(root);

    while (!q.empty()) {
        Node* cur = q.front();
        q.pop();

        cout << cur->data << ' ';

        if (cur->left != nullptr) q.push(cur->left);
        if (cur->right != nullptr) q.push(cur->right);
    }
}
```

### 배열로 표현한 완전이진트리

- 1-indexed: left = i * 2, right = i * 2 + 1
- 0-indexed: left = 2 * i + 1, right = 2 * i + 2

```cpp
void preorder(const vector<int>& tree, int idx) {
    if (idx >= (int)tree.size() || tree[idx] == -1) return;
    cout << tree[idx] << ' ';
    preorder(tree, idx * 2);
    preorder(tree, idx * 2 + 1);
}
```

## 2. Trie

> <u>트라이는 문자열을 문자 단위로 저장하는 트리다.</u>

- 접두사 검색에 강하다.
- 자동완성에 자주 쓴다.
- insert, search, startsWith가 기본이다.

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

struct TrieNode {
    bool isEnd = false;
    unordered_map<char, TrieNode*> next;
};

class Trie {
private:
    TrieNode* root;

public:
    Trie() : root(new TrieNode()) {}

    void insert(const string& word) {
        TrieNode* cur = root;
        for (char ch : word) {
            if (cur->next.count(ch) == 0) {
                cur->next[ch] = new TrieNode();
            }
            cur = cur->next[ch];
        }
        cur->isEnd = true;
    }

    bool search(const string& word) {
        TrieNode* cur = root;
        for (char ch : word) {
            if (cur->next.count(ch) == 0) return false;
            cur = cur->next[ch];
        }
        return cur->isEnd;
    }

    bool startsWith(const string& prefix) {
        TrieNode* cur = root;
        for (char ch : prefix) {
            if (cur->next.count(ch) == 0) return false;
            cur = cur->next[ch];
        }
        return true;
    }
};
```

```java
import java.util.HashMap;
import java.util.Map;

class TrieNode {
    boolean isEnd;
    Map<Character, TrieNode> next = new HashMap<>();
}

class Trie {
    private final TrieNode root = new TrieNode();

    void insert(String word) {
        TrieNode cur = root;
        for (char ch : word.toCharArray()) {
            cur = cur.next.computeIfAbsent(ch, key -> new TrieNode());
        }
        cur.isEnd = true;
    }

    boolean search(String word) {
        TrieNode cur = root;
        for (char ch : word.toCharArray()) {
            TrieNode next = cur.next.get(ch);
            if (next == null) return false;
            cur = next;
        }
        return cur.isEnd;
    }

    boolean startsWith(String prefix) {
        TrieNode cur = root;
        for (char ch : prefix.toCharArray()) {
            TrieNode next = cur.next.get(ch);
            if (next == null) return false;
            cur = next;
        }
        return true;
    }
}
```

## 3. 빈출 문제

### 1) 트리 순회

출제 스타일:

- 전위 / 중위 / 후위 결과 쓰기
- 주어진 순회 결과로 트리 복원
- 코드 구현

핵심:

- 전위: Root -> Left -> Right
- 중위: Left -> Root -> Right
- 후위: Left -> Right -> Root
- BFS는 레벨 순회

### 2) BST

자주 묻는 것:

- BST 삽입 / 탐색
- BST 검증
- 중위 순회가 왜 정렬인지

핵심:

- 왼쪽 < 루트 < 오른쪽
- 중위 순회 결과 = 오름차순

### 3) 높이 / 깊이 / LCA

자주 묻는 것:

- 트리 높이 구하기
- 노드 깊이 구하기
- 최저 공통 조상(LCA)

핵심:

- 깊이: 루트 기준
- 높이: 아래 리프까지
- LCA는 DFS나 부모 추적으로 구한다

### 4) 트라이

자주 묻는 것:

- 접두사 검색
- 전화번호 목록
- 자동완성
- 문자열 중복 판별

핵심:

- isEnd가 있어야 단어 끝을 구분한다.
- 알파벳 소문자면 배열, 일반 문자열이면 해시맵을 쓴다.
