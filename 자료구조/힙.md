# 힙

## 한 줄 요약

> <u>힙은 최댓값 또는 최솟값을 빠르게 꺼내기 위한 완전 이진 트리다.</u>

힙은 완전 이진 트리 형태를 유지하면서, 부모와 자식 사이의 우선순위 규칙을 만족하는 자료구조다.

- 최대 힙: 부모 >= 자식
- 최소 힙: 부모 <= 자식

힙은 정렬된 트리가 아니라 최댓값 / 최솟값을 빠르게 꺼내기 위한 구조다.

## 1. 핵심 특징

> <u>힙은 정렬된 트리가 아니라 우선순위가 보장되는 트리다.</u>

- 항상 완전 이진 트리여야 한다.
- 루트에 최댓값 또는 최솟값이 온다.
- 삽입과 삭제가 빠르다.
- 내부 전체가 정렬되지는 않는다.

배열로 표현하면 보통 이렇게 쓴다.

- 0-indexed: parent = (i - 1) / 2
- left = 2 * i + 1
- right = 2 * i + 2

## 2. 기본 코드

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    priority_queue<int> maxHeap;
    priority_queue<int, vector<int>, greater<int>> minHeap;

    maxHeap.push(3);
    maxHeap.push(1);
    maxHeap.push(5);

    cout << maxHeap.top() << '\n';
    maxHeap.pop();

    minHeap.push(3);
    minHeap.push(1);
    minHeap.push(5);

    cout << minHeap.top() << '\n';
    minHeap.pop();
}
```

```java
import java.util.PriorityQueue;
import java.util.Collections;

public class HeapExample {
    public static void main(String[] args) {
        PriorityQueue<Integer> minHeap = new PriorityQueue<>();
        PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());

        minHeap.offer(3);
        minHeap.offer(1);
        minHeap.offer(5);

        System.out.println(minHeap.peek());
        minHeap.poll();

        maxHeap.offer(3);
        maxHeap.offer(1);
        maxHeap.offer(5);

        System.out.println(maxHeap.peek());
        maxHeap.poll();
    }
}
```

## 3. 빈출 문제

### 1) 우선순위 큐

> <u>우선순위 큐는 힙으로 구현한다.</u>

- 가장 큰 값 / 작은 값 빨리 꺼내기
- 작업 스케줄링
- 이벤트 처리 순서 관리

핵심:

- 힙은 우선순위 큐 구현에 자주 쓴다.
- push, pop, top/peek가 기본이다.

### 2) K번째 수

> <u>K개만 유지하는 문제는 힙이 특히 잘 맞는다.</u>

- 가장 큰 값 K개 유지
- 가장 작은 값 K개 유지

핵심:

- 최소 힙과 최대 힙을 상황에 맞게 고른다.
- N개 중 K개만 계속 유지하면 효율적이다.

### 3) 힙 정렬

> <u>힙 정렬은 루트에서 하나씩 꺼내는 방식이다.</u>

- 배열을 정렬하는 방법
- 정렬 안정성 여부

핵심:

- 루트에서 하나씩 꺼내면 정렬 순서가 된다.
- 시간 복잡도는 O(N log N)이다.

### 4) 배열 인덱스 계산

> <u>배열 인덱스 규칙은 완전 이진 트리 문제의 기본이다.</u>

- 부모와 자식 인덱스 계산
- 완전 이진 트리 배열 표현

핵심:

- parent = (i - 1) / 2
- left = 2 * i + 1
- right = 2 * i + 2
