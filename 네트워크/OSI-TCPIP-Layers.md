# [네트워크] OSI 7 Layer / TCP/IP 계층

## 한 줄 요약

> <u>`OSI 7 Layer`는 네트워크를 7계층으로 나눠 설명하는 모델이고, `TCP/IP`는 인터넷 프로토콜 스택 관점의 실무적 계층 모델이다.</u>

OSI는 학습/설명용에 강하고, TCP/IP는 실제 구현 관점에서 많이 쓴다.

핵심 키워드: `L2`, `L3`, `L4`, `IP`, `TCP`, `HTTP`

<br />
<br />

---

## 1. OSI 7 Layer

- L1 Physical
- L2 Data Link
- L3 Network
- L4 Transport
- L5 Session
- L6 Presentation
- L7 Application

<br />
<br />

---

## 2. TCP/IP (대표적 4계층 표현)

- Link
- Internet (`IP`)
- Transport (`TCP/UDP`)
- Application (`HTTP` 등)

<br />
<br />

---

## 3. 자주 나오는 포인트

- “HTTP는 L7”, “TCP는 L4”, “IP는 L3”로 묻는 문제가 많다.
