# Contributing

## 문서 작성 기준

- 문단은 짧게 끊는다.
- 섹션 사이에는 여백을 둔다.
- 코드와 설명은 붙이지 말고 한 줄 띄운다.
- 백틱은 단어, 용어, 명령어, 함수명처럼 정확히 찍어야 하는 것에만 쓴다.
- 문장 전체를 강조할 때는 밑줄을 우선한다.
- `**굵은 강조**`는 짧은 결론, 정의, 비교 결과처럼 바로 보아야 하는 문장에 쓴다.
- `<u>밑줄</u>`은 문장 전체를 강조해야 할 때만 쓴다.
- 세 가지를 한 문장에 겹쳐 쓰지 않는다.
- 같은 페이지 안에서는 강조 방식이 흔들리지 않게 유지한다.

## 커밋 기준

- 커밋 본문에는 커밋 시간 설명을 적지 않는다.
- 시간 조정은 예외적으로 작업 지시가 있을 때만 따른다.
